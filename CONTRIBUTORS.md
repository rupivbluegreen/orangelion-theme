# Contributors

Thank you to everyone who helps make OrangeLion better. OrangeLion is an unofficial,
community-maintained CSS theme extension for Apache Guacamole, and it exists because of
volunteer effort.

**Maintainer: @rupivbluegreen**

## We value non-code contributions

You do not need to write CSS to help. Contributions the project genuinely values include:

- Screenshot refreshes when the interface or theme changes
- Documentation improvements, corrections, and examples
- Testing on specific Apache Guacamole versions and reporting what works
- Translations and reviews of existing translation strings

## How to get listed

Open a pull request that adds your name (and an optional GitHub handle) to the list below,
or ask a maintainer to add you when your contribution is merged.
